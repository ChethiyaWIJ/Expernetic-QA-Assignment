# Java Programming Questions

This repository contains solutions for two Java programming questions.

##  Factorial Calculator

This program calculates the factorial of a non-negative integer entered by the user.

Example:


5! = 5 × 4 × 3 × 2 × 1 = 120


The program:

* Takes input using the Scanner class
* Checks if the number is non-negative
* Uses a method to calculate the factorial

##  Pattern Generator

This program prints a star pattern using nested loops.

Example output:

```

  ***
 ****
*****
 ****
  ***
   **
    *
    
```

## Technologies Used
